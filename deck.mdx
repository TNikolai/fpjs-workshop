### Famous Functors: Maybe

---

export default Layout

```js Things we can do with Maybe
Maybe.of('Malkovich Malkovich').map(match(/a/gi))
// Just(True)

Maybe.of(null).map(match(/a/gi))
// Nothing

Maybe.of({ name: 'Boris' })
  .map(prop('age'))
  .map(add(10))
// Nothing

Maybe.of({ name: 'Dinah', age: 14 })
  .map(prop('age'))
  .map(add(10))
// Just(24)
```
